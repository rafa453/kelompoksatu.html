Microsoft Excel

Microsoft excel merupakan adalah program aplikasi berbasis spreadsheet yang dikembangkan oleh Microsoft. Aplikasi ini dirancang untuk membantu pengguna mengolah data berupa angka, teks, atau grafik dalam bentuk tabel.

1. Left, Mid, Right

1. LEFT adalah salah satu fungsi teks yang digunakan untuk mengambil sejumlah karakter tertentu dari bagian paling kiri sebuah teks atau string. Menggunakan rumus : =LEFT(teks, [jumlah_karakter])

2. MID adalah fungsi yang digunakan untuk mengambil bagian tertentu dari teks atau string, dimulai dari posisi tertentu, dan sepanjang jumlah karakter yang ditentukan. menggunakan rumus : =MID(teks, posisi_awal, jumlah_karakter)

3. RIGHT adalah fungsi yang digunakan untuk mengambil sejumlah karakter tertentu dari bagian paling kanan sebuah teks atau string. Menggunakan rumus : =RIGHT(teks, [jumlah_karakter])

2. VLOOKUP dan HLOOKUP

1. VLOOKUP adalah fungsi dalam Microsoft Excel yang digunakan untuk mencari nilai tertentu di kolom pertama suatu tabel dan mengembalikan nilai dari kolom lain yang sesuai di baris yang sama. Menggunakan rumus : =VLOOKUP(Nilai Kunci; Tabel Referensi; Kolom yg digunakan; 0)

2. HLOOKUP adalah fungsi dalam Microsoft Excel yang digunakan untuk mencari nilai tertentu di baris pertama suatu tabel dan mengembalikan nilai dari baris lain yang sesuai dalam kolom yang sama, fungsinya mirip dengan VLOOKUP, tetapi bekerja secara horizontal (mencari data dalam baris, bukan kolom). Menggunakan rumus : =HLOOKUP(Nilai Kunci; Tabel Referensi; Kolom yg digunakan; 0)

3. IF

IF Berfungsi untuk logika yang digunakan untuk melakukan pengujian terhadap suatu kondisi tertentu dan mengembalikan nilai yang berbeda berdasarkan hasil pengujian tersebut.

2.1 IF menggunakan rumus : =IF(logical_test, value_if_true, value_if_false)

2.2 IF Bertingkat Menggunakan rumus : =IF(kondisi1, hasil1, IF(kondisi2, hasil2, IF(kondisi3, hasil3, hasil_lain)))

2.3 IF AND, AND akan mengembalikan TRUE jika semua kondisi terpenuhi, dan FALSE jika salah satu kondisi tidak terpenuhi. Menggunakan rumus : =IF(AND(kondisi1, kondisi2, ...), nilai_jika_true, nilai_jika_false)

2.4 IF OR Kombinasi keduanya berguna ketika Anda ingin memeriksa apakah salah satu dari beberapa kondisi terpenuhi. Menggunakan rumus : =IF(OR(kondisi1, kondisi2, ...), hasil_jika_benar, hasil_jika_salah)

4. Aritmatika

Aritmatika bagian dari matematika yang mempelajari tentang operasi dasar pada angka, seperti penjumlahan, pengurangan, perkalian, dan pembagian

1.1

Simbol Keterangan
+ Pertambahan
- Pengurangan
* Perkalian
/ Pembagian
^ Pembagian

1.2 SUM

Sum digunakan untuk menjumlahkan angka-angka dalam satu atau lebih sel. Menggunakan rumus : =SUM(number1, [number2],...)

1.3 Average

Average digunakan untuk menghitung rata-rata aritmatika dari satu atau lebih angka. Menggunakan rumus : =AVERAGE(number1, [number2],...)

1.4 Min

Min digunakan untuk mencari nilai terkecil dari sekumpulan angka atau rentang sel. Menggunakan rumus : =MIN(number1, [number2],...)

1.5 Max

Max digunakan untuk mencari nilai terbesar dari sekumpulan angka atau rentang sel. Menggunakan rumus : =MAX(number1, [number2],...)

1.6 Rank

Rank digunakan untuk menentukan peringkat atau urutan suatu angka dalam sebuah rentang data, berdasarkan urutan nilai. Menggunakan rumus : =RANK(number, ref, [order]), Catatan : Klik Fn+F4 atau F4 untuk mengunci nilai.

1.7 Count

Count digunakan untuk menghitung jumlah sel yang berisi angka dalam sebuah rentang data. Menggunakan rumus : =COUNT(value1, [value2],...)

1.8 CountBlank

CountBlank digunakan untuk digunakan untuk menghitung jumlah sel kosong dalam sebuah rentang data. Menggunakan rumus : =COUNTBLANK(range)
