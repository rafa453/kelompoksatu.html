Microsoft Word

Microsoft Word adalah program aplikasi pengolah kata yang berfungsi untuk membuat, menyunting, dan memformat dokumen.

1. ShortCut Pada microsoft word

  • Bold/Tebal = Ctrl + B

  • Italic/Miring = Ctrl + I

  • Underline/Garis Bawah = Ctrl + U

  • Center/Rata Tengah = Ctrl + E

  • Left/ Rata Kiri = Ctrl + L

  • Right/Rata Kanan = Ctrl + R

  • Justify/Rata Kiri Kanan = Ctrl + J

  • Open/Buka = Ctrl + O

  • Save/Simpan = Ctrl + S

  • Close/Tutup = Ctrl + W

  • Cut/Potong = Ctrl + X

  • Copy/Salin = Ctrl + C

  • Paste/Tempel = Ctrl + V

  • Select All/Pilih Semua = Ctrl + A

2. DropCap

    Drop cap adalah huruf kapital besar atau kombinasi karakter yang digunakan sebagai elemen dekoratif di awal paragraf atau bagian.

    2.1 Cara Mengaplikasikannya

    1. Pada Tab Insert, Pilih DropCap
